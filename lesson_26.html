Синтаксис django шаблонов

# Команда Что делает
1 {% extends "name.html" %} Встраивает контент из шаблона
2 {% include 'navbar.html' %} Добавляет контент из другого шаблона
3 {% block name %} {% endblock %} Резервирует место для контента из других шаблонов
4 {% for foo in bar %} {% endfor %} Инициирование цикла
5 {% if foo %} {% elif bar %} {% else %} {% endif %} Инициирование ветвлений

Синтаксис для работы с объектами из view модуля

# Команда Что делает
1 Model_name.objects.all() Возвращает итератор со всеми записями из модели (базы данных)
2 Model_name.objects.get(id=1) Возвращает объект из модели (базы данных)
3 return HttpResponse('Hello world') Возвращает html страницу с контентом
4 return render(request, 'template.html', {'foo': bar}) Возвращает отображение html шаблона, где используется контекст {'foo': bar}
