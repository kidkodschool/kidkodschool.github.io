Установка и настройка git

  1. Скачать и установить git на компьютер

  2. Откройте терминал (cmd)

  3. Откройте терминал и используйте следующую команду, чтобы добавить своё имя: git config --global user.name "ваше имя"

  4. Для добавления почтового адреса вводите: git config --global user.email "адрес"

  5. Для добавления вашего пароля введите: git config --global user.password "пароль"
