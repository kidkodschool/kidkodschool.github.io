Ответ на задания должен содержать не более 1 строчки кода

  1. Функция получает на входе: Список из чисел. Задание вернуть среднее арифметическое значение элементов из этого списка. Например: [1, 5, 87, 45, 8, 8] == 25

  2. Функция получает на входе: Направление - d, и список - a. Задание вернуть отсортированный список в зависимости от направления. R - от меньшего к большему, L - от большего к меньшему. Например: 'L', [1, 4, 5, 3, 5] == [5, 5, 4, 3, 1]

  3. Функция получает на входе количество: h - часов, m - минут, s - секунд. Задание вернуть общую сумму времени в миллисекундах. Например: 1,1,1 == 3661000

  4. Функция получает на входе: Число - num. Задание вернуть сумму всех составляющих этого числа. Например: 4 == 10, потому что 1 + 2 + 3 + 4 = 10

  5. Функция получает на входе: Количество здоровья персонажа - health. Задание вернуть булев тип для определения его состояние (0 или меньше False). Например: -2 == False

  6. Функция получает на входе: Список - arr. Задание вернуть наименьшее уникальное положительное число из возможных. Например: [0, 1, 2, 3, 5] == 4
