Веб запросы

# Тип запроса Что делает
1 GET Получение данных с сервера
2 POST Отправка данных на сервер (например через форму)
3 DELETE Удаление файла с сервера
4 PUT Создание файла на сервере
5 HEAD Получение данных с сервера (без основных данных вроде index.html)
6 CONNECT Создание http связи между клиент-сервером
7 OPTIONS Запрос на получение всех доступных методов у сервера
8 TRACE Запрос используемый для процесса дебага
