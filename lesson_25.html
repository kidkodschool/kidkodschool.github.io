Необходимые команды для работы с django проектом

# Команда Что делает
1 pip install django==3.1.5 Используемая нами версия django
2 django-admin Список всех доступных команд
3 django-admin startproject project_name Создание базового приложения
4 django-admin startapp app_name Добавление "модуля" к приложению
5 py manage.py runserver Запуск приложения
6 py manage.py makemigrations Добавление в очередь для обновления базы данных. Важно! Выполняется до migrate
7 py manage.py migrate Обновление базы данных
8 py manage.py createsuperuser Добавление административного аккаунта в приложение. Важно! Запускать перед makemigrations и migrate
9 py manage.py shell Запуск интерпретатора для работы с базой данных через консоль
