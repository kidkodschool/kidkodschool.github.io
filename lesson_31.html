Пошаговая инструкция размещения приложения на Heroku

  1. Создать аккаунт на heroku

  2. Создать новый репозиторий на github или использовать ранее созданный

    Подробная инструкция

  3. Создать файл .gitignore и добавить в него следующий контент

    # Text backup files
    *.bak

    # Database
    *.sqlite3

  4. Создать файл runtime.txt и добавить в него следующий контент

    python-3.8.6

  5. Создать файл Procfile и добавить в него следующий контент

    web: gunicorn locallibrary.wsgi --log-file -

  6. Установить gunicorn

    pip install gunicorn

  7. Установить dj-database-url

    pip install dj-database-url

  8. Установить whitenoise

    pip install whitenoise

  9. Открыть settings.py и добавить в него следующий контент (в самом конце файла)

    # Heroku: Update database configuration from $DATABASE_URL.
    import dj_database_url
    db_from_env = dj_database_url.config(conn_max_age=500)
    DATABASES['default'].update(db_from_env)

  10. Открыть settings.py и изменить/добавить в него следующий контент

    STATIC_ROOT = BASE_DIR / 'staticfiles'
    STATIC_URL = '/static/'

  11. Открыть settings.py и добавить в MIDDLEWARE после 'django.middleware.security.SecurityMiddleware' следующую строчку

    'whitenoise.middleware.WhiteNoiseMiddleware'

  12. Открыть settings.py и изменить/добавить в него следующий контент

    STATICFILES_STORAGE = 'whitenoise.storage.CompressedManifestStaticFilesStorage'

  13. Создать requirements.txt

    pip3 freeze > requirements.txt

  14. Добавить все файлы в github репозиторий

    git add .
    git commit -m "init"
    git push

  15. Создать новый контейнер на heroku и отправить туда файлы

    heroku login
    heroku create
    git push heroku main

  16. Проверить, что все работает

    heroku open

  17. *Вспомогательные команды на heroku

    heroku run python manage.py migrate
    heroku run python manage.py createsuperuser
